Qualquer texto para nome de cores

Os valores vão de 0 a 9 e A a F. O 0 é o mais escuro e F o mais claro

O valor #000 é a cor preta e o valor #FFF é a cor branca

Temos que inserir 6 digitos, precedidos de uma #

Se um valor for repetido 6 vezes, podemos escrever abreviada. Por exemplo: ao invés de #FFFFFF, podemos escrever #FFF

A mesma coisa vale para #112233 que pode ser #123

Os valores vão de 0 a 255. O 0 é o mais escuro e o 255 é o mais claro.

A sintaxe é (0-255, 0-255, 0-255)

O primeiro representa o vermelho, depois verde e por fim azul.

Para criar o verde usamos (0, 255, 0)

O RGBA é a mesma coisa que o rgb, a diferença é que o A (de alpha) altera a opacidade da cor.

Os valores possíveis são de 0 a 1 (vem de porcentagem, por exemplo: 0.1 representa 10%)

Sendo assim, o 0 é transparete e 1 totalmente visível.

HSL é um acrômio para hue, saturarion e lightness

Ela não é muito utilizada

Podemos definir a cor com: hsl(0-255, 0-100%, 0-100%)

Quase todos os elementos tem um fundo, podemos manipular as cores deles com o background color.

A regra para colocar a cor de fundo é backgroung-color: "cor";

As regras de cores utilizadas podem ser a que vimos anteriormente.

Podemos utilizar o opacity: "valor"; para alterar a opacidade de um elemento.

Os valores são de 0 a 1

Se utilizar o opacity, todos os elementos filhos também terão a opacidade alterada.

Caso queira mudar a opacidade somente do fundo, podemos utilizar o rgba na cor de fundo.

Podemos colocar imagem no fundo com a regra background-image: url("pasta/imagem,jpg")

Geralmente a imagem fica em outra pasta, então temos que voltar um diretório utilizando o símbolo ..

Podemos também centralizar a imagem de fundo

Para isso utilizamos 2 regras, sendo elas Background-position com o valor center.

A segunda regra é background-size com o valor de cover.
